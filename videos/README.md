## 视频规格要求

- **格式**：MP4 (H.264 编码)
- **分辨率**：1920x1080 或 1280x720
- **码率**：建议 2-5 Mbps（平衡质量和加载速度）
- **时长**：10-30 秒循环片段
- **文件大小**：单个文件建议 < 10MB
- **音频**：建议静音或轻音乐背景

## 上传后

视频上传完成后，网页会自动加载这些视频作为背景，无需修改代码。
